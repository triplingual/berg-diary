About “Locating Mary Berg’s Diary”

When it is ready, a comment on creating this project will appear here. Until then, this page contains links to the project's files, including the geoJSON data file, and the tools and resources used to create it.

  • GitHub Repository
  • geoJSON in a gist
  • Maps: National Library of Poland (for specific links, see the main interface map credits)
  • Georectification: MapWarper
  • Initial geospatial data plotting and some later editing: GeoJSON.io
  • Mapmaking with Mapbox.js 2.4.0 and Leaflet
  • Date formatting with Moment.js
  • Simple (and common) styling with Bootstrap
  • The essay in the primary interface contains no footnotes or bibliography to simplify the readability and the markup. However, you can access the scholarly version [PDF] as well.
